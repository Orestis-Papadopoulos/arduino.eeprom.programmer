An adaptation of Ben Eater's EEPROM programmer (https://www.youtube.com/watch?v=K88pgWhEb1M).

Microcontroller: Arduino Nano <br />
EEPROM: AT28C64B (he uses the 28C16) <br />
Shift registers: SN74HC595 (he uses the 74HC595) <br />

Note: the registers I used have a different pinout than the ones used by Ben. Specifically, the shift clock and storage clock are vice versa.
